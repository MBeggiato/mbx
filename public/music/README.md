# Music Files

This folder contains your music library for the music player.

## Supported Formats:

- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)
- M4A (.m4a)

## How to Add Music:

1. Copy your music files into this folder
2. The music player will automatically detect them
3. Organize them in subfolders if desired
